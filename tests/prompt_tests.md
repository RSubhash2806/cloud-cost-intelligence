# Prompt Testing Documentation

## Test Cases

### Query Type: Cost Analysis
- **Test 1**: Analyze total monthly costs
  - Query: "What are our total cloud costs for the current month?"
  - Expected: Summary with breakdown by service

- **Test 2**: Service comparison
  - Query: "Compare compute vs storage costs"
  - Expected: Comparative analysis with percentages

### Query Type: Optimization
- **Test 3**: High-cost resources
  - Query: "What are our top 10 most expensive resources?"
  - Expected: Ranked list with cost details

- **Test 4**: Anomaly detection
  - Query: "Identify unusual cost spikes"
  - Expected: List of anomalies with explanations

## Evaluation Metrics
- Response relevance score
- Factual accuracy
- Response completeness
